Property Toolbox

When the Property Toolbox option is checked, the property toolbox is displayed on the right side of the screen. When it is unchecked, the property toolbox is hidden.

Log

When the Log option is checked, the log view is displayed at the bottom of the screen. When it is unchecked, the log view is hidden.
